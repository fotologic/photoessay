See the essays your students have shared with you, set milestones, leave feedback, and sign work off for publication.

The teacher view asks for full Drive access so it can find the essay folders students share with you and write feedback into them, with no file picker friction. Students' own app uses the narrow drive.file permission. In a school deployment this choice gets revisited with the network manager.

Milestones are written into each student's shared essay folder as milestones.json, so their timeline updates next time they open PhotoEssay. Reordering or removing milestones resets any per-student "reviewed" ticks, so settle the list early in the year.

Each card is a folder a student has shared with you from their own Google Drive. Their essay stays theirs; your feedback, milestones and sign-off are separate files written alongside it, and this page stores nothing anywhere else. Drive keeps version history on everything.
